## What is Sol?

Sol is an open-source LLM designed to be **lightweight, accessible, and transparent**. Built by a small team of developers who believe powerful AI shouldn't require enterprise-grade hardware to run.

The goal is simple — a model that anyone can run, modify, and contribute to.

---

## ✨ Key Goals

- **Lightweight** — Designed to run on consumer hardware with minimal resources
- **Open** — Fully open-source, every weight, every line of code
- **Community-driven** — Built by real people, not a corporation
- **Transparent** — No black boxes, no mystery
